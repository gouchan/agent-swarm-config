### Start a Swarm
```
/swarm "Build a complete authentication system"
```

### Run a Pipeline
```
/pipeline architect → executor → qa-tester → security-reviewer
```

### Create Video
```
/remotion "Create a 30-second product demo with text animations"
```

### Deep Research
```
/perplexity "What are the App Store review guidelines for 2024?"
/deepsearch "How does the payment flow work in this codebase?"
```

### Autonomous Mode
```
/autopilot "Build a landing page with newsletter signup"
/ralph "Create a full CRUD API for user management"
```
